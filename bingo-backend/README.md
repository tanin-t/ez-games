# bingo-game-backend

## Setup
```
# create virtualenv
python -m virtualenv venv

# activate virtualenv
source venv/bin/activate

# install dependencies
pip install -r requirements.txt

# migrate db
python manage.py migrate

# runserver
python manage.py runserver
```
